name: currency display standard
description: 모든 통화 표시는 USD 기준, KRW는 보조로 표시
type: feedback
---

모든 화면에서 통화는 USD를 주(primary)로 표시하고, KRW(원화)는 작은 글씨로 보조(secondary) 표시한다.

**Why:** 사용자 명시 요청 — 홈, 계산기, 히스토리 순서로 적용하며 "앞으로 모든 통화는 달러를 기준으로"라고 명확히 지정함.

**How to apply:** 새 화면이나 수치 표시 추가 시 항상 USD 먼저, 그 다음 `≈ KRW` 형식으로 보조 표시. KRW만 단독으로 표시하지 말 것.
